# Projet-Systeme
Projet de M1 de Système.
Le but du script est d'analyser un fichier texte contenant le spectre d'une photodiode.
Concrètement, le fichier associe pour chaque longueur d'onde une intensité.
Le script récupère ces informations et en demandant une taille de fenêtre de longueur d'onde et un intervalle souhaité affiche le graphe des intensités moyennes sur chaque plage de longueur d'onde sur l'intervalle demandé.
